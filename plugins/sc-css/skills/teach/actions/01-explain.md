# Action 01 — explain

## Rôle

Expliquer un concept CSS en ancrant l'explication dans le code du projet courant.

## Procédure

1. Identifier le concept demandé.
2. Chercher dans le projet un exemple réel du pattern (ou de son absence) — grep CSS.
3. Expliquer le principe sur l'exemple trouvé.
4. Si le projet n'a pas d'exemple pertinent, utiliser un exemple générique minimal.
5. Proposer une application concrète au projet si pertinente (sans éditer).

## Format

Réponse directe, sans en-têtes de section sauf si l'explication est longue (> 3 points).
Code en blocs ` ```css ` avec la ligne du projet référencée si applicable.
