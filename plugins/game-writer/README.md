# game-writer

*Écriture de contenu narratif pour le jeu vidéo (projet 8-MINE) : écriture et revue de timelines dialogiques, et gestion de la bank d'assets narrative.*

Plugin orienté **création de contenu** narratif de jeu vidéo (conventions et chemins propres au projet 8-MINE) : registre de ressources, pipeline d'écriture de scènes dialogiques et revue du graphe narratif. La **partie technique** (GDScript, moteur Godot) vit dans le plugin `sc-godot`. (ex-`gamedesign`)

## Skills

| Skill | Description |
|---|---|
| `bank` | Gère le registre de ressources 8-MINE (`bank.yml`) — initialisation et challenge de la bank d'assets narrative |
| `dialogic-draft` | Pipeline d'écriture séquentiel des timelines Dialogic 8-MINE (spec de scène, comportement PNJ, arcs, DTL) |
| `dialogic-review` | Revue et validation des timelines Dialogic et du graphe narratif (précheck, persona, audit de graphe, gestion des nœuds) |
