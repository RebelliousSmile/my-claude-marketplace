## Description

Implement password reset functionality that allows users to reset their password via email verification link. Users request reset, receive email with secure link, and can set new password.

## Context

Current system has no password recovery mechanism. Users who forget their password must contact support. This feature will reduce support tickets and improve user experience.

---

## Acceptance Criteria

- [ ] User can request password reset from login page
- [ ] System sends email with reset link within 30 seconds
- [ ] Reset link expires after 1 hour
- [ ] Reset link is single-use (invalidated after use)
- [ ] User can set new password meeting security requirements
- [ ] User receives confirmation email after successful reset
- [ ] Invalid/expired links show appropriate error message

---
