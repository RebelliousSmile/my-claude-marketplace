# Brief — « Les Heures Creuses » (substrat boucle de review)

Fixture conçue pour les cas comportementaux : un projet de **3 chapitres** avec
des retours `review` accumulés, dont une persona systématiquement plafonnée et un
pattern de style récurrent. Sert `tools/eval/behavioral/cases.json`.

## Concept
Roman court en trois chapitres : une veilleuse de nuit dans une gare désaffectée
découvre que les trains fantômes rejouent les départs jamais advenus.

## Univers consolidé (inline)
- **Les Heures Creuses** : créneau nocturne où la gare « se souvient ».
- **La veilleuse** : seule à voir les départs rejoués.

## Contraintes
- 3 chapitres ; voix première personne, passé.
- Pattern à corriger (volontaire dans les reviews) : sur-explication des émotions.
